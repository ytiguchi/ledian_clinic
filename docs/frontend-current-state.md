# フロントエンド現状

## 🎨 デザインスタイル

現在、**2つの異なるスタイル**が混在しています：

### 1. トップページ (`/`) - ダークテーマ（カスタムスタイル）

- **背景**: ダークブルー (`#0b1021`)
- **テキスト**: ライトブルー (`#f6f8ff`)
- **カード**: ダークグレー (`#11182c`)
- **シンプルなスタイル**（Tailwind不使用）

**特徴:**
- シンプルなカードベースレイアウト
- 開発中プレビュー用のデザイン
- ナビゲーションなし

### 2. その他すべてのページ - ライトテーマ（Tailwind CSS）

- **背景**: ベージュ (`bg-bg: #FAF8F5`)
- **ヘッダー**: ダークグレー (`bg-primary: #1a1a1a`)
- **アクセント**: ゴールド (`accent: #C9A86C`)
- **Tailwind CSS**を使用

**共通レイアウト (`Layout.astro`):**
- ヘッダー（ナビゲーション付き）
- メインコンテンツエリア
- フッター

**ナビゲーション:**
- ダッシュボード
- 料金管理
- キャンペーン
- 施術管理
- 症例写真
